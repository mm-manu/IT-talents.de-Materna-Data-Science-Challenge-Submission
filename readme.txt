Requirements:
- docker (https://docs.docker.com/install/)
- docker-compose (https://docs.docker.com/compose/install/)

Usage:
1. Run docker-compose up (needs root and internet connectivity)
2. Open a web browser and locate to localhost:8888
3. Select races_analysis.ipynb
4. In the top menu go to "Cell" -> "Run all"

It may take a couple of minutes until all cells have finished.
Shutdown with Ctrl+c.
